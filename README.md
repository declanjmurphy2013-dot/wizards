# Wizards & Warriors

A high-fantasy browser fighting game inspired by classic arena fighters, built as a single-file web game.

---

## 🎮 Overview
Wizards & Warriors is a fast-paced 1v1 fighting game featuring unique characters, abilities, and dynamic combat. Play against CPU or challenge friends locally using a generated code.

---

## 🧙 Characters
- 12 unique fighters
- Each has:
  - Custom stats (Attack, Defense, Speed)
  - Unique weapon and abilities
  - Signature moves
- Includes **Morvane, Lord of the Undead (Necromancer)**

---

## ⚔️ Controls
| Key | Action |
|-----|--------|
| Arrow Keys | Move / Jump |
| Z | Block |
| V | Quick Attack |
| X | Heavy Attack |
| C | Signature Move |
| Shift | Grab / Throw |
| Space | Super Move |

---

## 🔥 Gameplay Features
- Health + Super meter system
- Hidden combos (not shown to player)
- Character stat balancing
- Difficulty modes:
  - Easy
  - Normal
  - Hard
  - Nightmare
- Smooth animations (run, jump, combat)
- Unique character models and weapons

---

## 🌐 Multiplayer (Local)
- Host / Join system using 6-digit code
- Uses browser BroadcastChannel (same machine or browser environment)

---

## 🎵 Audio System
- Dynamic music engine
- Modes:
  - Menu (ambient)
  - Battle (action)
  - End (win/lose)
- Low-health visual intensity (red flashing border)
- Music adapts to game state
